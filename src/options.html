VK-Pause

Приложение для лёгкого управления проигрыванием музыки в ВК

Переключайте композиции двойным кликом по иконке приложения (или по двойному нажатию горячей комбинации ALT+N при активном окне… Переключайте композиции двойным кликом по иконке приложения (или по двойному нажатию горячей комбинации ALT+N при активном окне браузера, а также Ctrl+Shift+9 при неактивном, или скрытом). Ctrl+Shift+9 -- старт/стоп/переключение композиции из ВК даже ВНЕ браузера! Приложение работает с НОВЫМ и старым дизайном ВКонтакте. Удобное расширение для управления (старт-стоп-следующая) Вашей музыкой в VK. Работает со всеми открытыми окнами и вкладками. Поддерживает несколько запущенных плееров (работает с последним прослушиваемым). Приложение гарантировано не передаёт Ваши личные данные третьим лицам, и не выполняет функций свыше описанных в подробном описании. Спасибо за установку, и слушайте #аудиокниги#: они сделают Вас умнее!:
